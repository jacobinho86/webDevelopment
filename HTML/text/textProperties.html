Bold & Italic

By enclosing words in the tags < b > and </b> you can characters appear bold. For example, to show the word important in bold you'll code it like so important, being important the result. By enclosing words in the tags < i > and </i > will make the characters appear italic, for example, the word ipsium encoded with the markup ipsium.

Superscript & subscript

The element < sup > is used to show the characters in between as a superscript, like the nth power of two will be coded as 2 < sup > n < /sup >, so the browser will show 2n. On the other hand the element < sub > is used to show characters as subscripts, like the chemical formula of water can be coded as H < sub > 2 < \sub > O, will be shown as H2O.

Citations & Definitions

When you are referencing a piece of work use the < cite > element to indicate where the citation is from. Browsers will render the content of a < cite > elements in italics. For example, < cite > make it or break it < /cite >, will be shown as make it or break it.

The < dfn > element is used to indicate the defining instance of a new term. Some browsers show the content of the < dfn > element in italics. For example, " < dfn > point </dfn > in brief a point is a point " is shown as: point in brief a point is a point.
